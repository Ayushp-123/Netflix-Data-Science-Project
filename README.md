# 🎬 Netflix Data Science Project

## 📌 Project Overview

This is an end-to-end Data Science project built using the Netflix Titles Dataset.

The project demonstrates the complete Data Science workflow including:

- Data Cleaning
- Data Visualization
- Exploratory Data Analysis (EDA)
- Machine Learning
- Business Analytics

The main objective of this project is to analyze Netflix content trends and build predictive models using Python and Machine Learning techniques.

- Feature Selection

- Model Training
- Prediction
- Accuracy Evaluation
- Confusion Matrix Visualization

# 💼 4. Real-world Business Analytics Project

Performed business-oriented analysis to identify:
- Audience preferences
- Netflix growth trends
- Most dominant genres
- Business insights from streaming data

Business Insights:
- Netflix contains significantly more Movies than TV Shows.
- USA contributes the highest amount of content.
- TV-MA is the most common rating category.
- Netflix growth accelerated after 2015.
- International and Drama genres dominate the platform.

Notebook:
```plaintext
netflix_business_project.ipynb
```
